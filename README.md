# Retail Store Service Operations: SQL Portfolio Project

This project is about ecommerce operations daily service report that shows all the customers data regarding their purchase, including status and various stages of production and so on.

Tables

- 'customers' — stores customer profiles and their order information
- 'service_requests' — tracks every service case and its current status
- 'parts_orders' — tracks individual parts ordered for each service case

---

Queries

The queries in this project answer real operational business questions:

- Current service ticket cases and their status
- Customers who require a technician to visit and complete the service request
- Customers whose general information is missing
- All scheduled appointments ordered by date
- Open cases that have not been scheduled yet
- Finding which assigned employee is handling what tickets

---

Goal

My goal is to build business and tech knowledge by combining operations, management and customer skills, and solving problems while learning and implementing SQL queries to simplify the process and make gathering requirements efficient.
